---
title: Downloading by version
description: How to let the Gradle Jextract plugin download and use Jextract depending on a given JVM target version
sidebar:
  order: 2
---

The plugin downloads and installs Jextract from the [official release page][jextract] for the current build platform.
It infers the matching Jextract version from the Java toolchain configuration of the Java plugin.

:::caution[modern JDK]
Your target JVM should be of version 19 or above.
:::

<Tabs syncKey="dsl">
<TabItem label="Kotlin">

```kts "languageVersion" title="Configuring the Jextract version via Java plugin dSL"
// build.gradle.kts
java {
    toolchain {
        languageVersion = JavaLanguageVersion.of(25)
    }
}
```

</TabItem>
<TabItem label="Groovy">

```groovy "languageVersion" title="Configuring the Jextract version via Java plugin dSL"
// build.gradle
java {
    toolchain {
        languageVersion = JavaLanguageVersion.of 25
    }
}
```

</TabItem>
</Tabs>

[jextract]: https://jdk.java.net/jextract/
